Steps to Download the packages

**Windows:**
pip install -r requirements.txt

**macOS:**
pip3 install -r requirements.txt

**Linux:**
# System dependencies first
sudo apt-get update
sudo apt-get install python3-tk python3-pip build-essential
 
pip3 install -r requirements.txt
